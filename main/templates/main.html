Moods

Moods is a full stack application created using Django framework version 2.2.4 as a final project with the Coding Dojo and Axsos Academy. The idea behind Moods is to connect people in different time zones, and it mimics most of the functionality of Facebook. Users can create their own profiles with random avatars, post on the main dashboard, comment, message other users, and view the time for users in different time zones.
